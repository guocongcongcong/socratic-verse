# AI Agent 开发学习路线

> 从零基础到能构建生产级 Agent 系统的完整学习路径。三层递进，每层对应不同的教法和深度。

---

## 三层递进结构

```
┌─────────────────────────────────────────────────────┐
│  入门篇（5+3 篇）  理解 Agent → 写 Skill → 工具 MCP  │
│  ├─ 目标：理解 Agent 思维方式，能写 Skill           │
│  ├─ 时间：2-4 周                                     │
│  └─ 教法：V1 苏格拉底追问                            │
├─────────────────────────────────────────────────────┤
│  进阶篇（8 篇）    构建 Agent → 框架 → 评估          │
│  ├─ 目标：能用框架构建 Agent，理解评估体系           │
│  ├─ 时间：4-8 周                                     │
│  └─ 教法：V2 费曼 + V5 合书复述                      │
├─────────────────────────────────────────────────────┤
│  高级篇（5 篇）    自主系统 → 多Agent → 生产化       │
│  ├─ 目标：能设计和部署生产级 Agent 系统              │
│  ├─ 时间：8-12 周                                    │
│  └─ 教法：V7 对子互教 + V8 自适应                   │
└─────────────────────────────────────────────────────┘
```

## 如何使用 Socratic Verse 学习

在 Claude Code 中启动后，根据你的层级选择教法：

```
# 入门篇（V1 苏格拉底）
"用 V1 模式教我 materials/agent/01-入门篇/理解Agent.md"

# 进阶篇（V2 费曼）
"用 V2 费曼模式教我 materials/agent/02-进阶篇/Agent的记忆系统.md"

# 高级篇（V7 对子互教）
"用 V7 对子互教模式讨论 materials/agent/03-高级篇/多Agent协作与Swarms.md"
```
