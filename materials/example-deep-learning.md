## 第一章：什么是深度学习

深度学习是机器学习的一个子集。它使用多层神经网络从数据中学习层次化的特征表示。

传统的机器学习需要人工设计特征（feature engineering）——你需要告诉算法"看什么"。
深度学习自动学习特征——算法自己发现"看什么"。

举个栗子：识别一张猫的照片。
- 传统方法：你告诉算法看边缘、纹理、颜色分布
- 深度学习方法：你给算法100万张猫照片，它自己学会什么是猫

## 第二章：神经网络基础

神经网络的基本单元是"神经元"——一个数学函数，接收多个输入，产生一个输出。

```
输入1 ──→
输入2 ──→  [加权求和] → [激活函数] → 输出
输入3 ──→
```

关键概念：
1. **权重（Weight）**：每个输入的重要性
2. **偏置（Bias）**：平移激活函数的阈值
3. **激活函数（Activation）**：引入非线性——没有它，多层网络等于一层

## 第三章：为什么是"深度"

"深度"指的是网络的层数。每一层学习不同级别的特征：
- 第一层：边缘、颜色
- 第二层：形状、纹理
- 第三层：物体部件（眼睛、鼻子）
- 更深层：完整物体（猫的脸）

这种层次化的表示是深度学习强大的根本原因——复杂的概念由简单的概念组成。
