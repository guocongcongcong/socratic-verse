# AI Agent 开发学习路线

> 面向非程序员的 Agent 开发入门。不要求精通编程，但需要理解 Agent 的思维方式。

---

## 五层递进结构

```
第一层  理解 Agent → 概念 + 必读文章
第二层  写 Skill    → 从拆解到创造
第三层  工具与 MCP  → Agent 的手和脚
第四层  上下文工程  → Agent 的"视野"管理
第五层  驾驭工程    → 让 Agent 持续可靠工作
```

---

## 学习节奏

| 阶段 | 时间 | 内容 |
|------|------|------|
| 1 | 第1-2周 | 读完第一层所有文章，理解 Agent Loop |
| 2 | 第3-4周 | 写第一个 Skill（挑你最熟悉的场景） |
| 3 | 第5-6周 | 深入 Harness，读《马书》源码 |
| 4 | 第7-8周 | 设计自己的 Agent 系统 |
