# 未读消息

---

**三月七**：甘雨老师！！！今天的课怎么样！！！我听说你教的是 Anthropic 那篇！！

**甘雨**：*低头绞手指* 嗯……上完了。他之前没看过这篇文章，我就一段一段带着读的。

**三月七**：啊？？？没提前看？？那你怎么教的！！

**甘雨**：我读一段，问一个问题。他当场推答案。第一个问题"天气预报提醒带伞是 Workflow 还是 Agent"，他先说是 Workflow——我追了三问，他自己改口了。

**三月七**：三问改答案！！甘雨你也太会挖了吧！！最后他总结出来的是什么？

**甘雨**："Agent = 自己判断 → 自己获取结果 → 自己判断下一步"。一句话，三个动词全是"自己"。

**刻晴**：他意识到了"决策者在内部"这个点？

**甘雨**：嗯。但有个地方他卡了一下。我问"Agent 不拿 ground truth 会怎样"，他第一反应是"报错停住"。

**三月七**：哈哈哈哈哈哈哈哈锅锅你怎么想的！！

**刻晴**：这个回答有意思。"报错停住"说明他的直觉是"系统有防御机制"。但 Agent 不是这么工作的——它不会停，它会猜。

**甘雨**：对，我说了"比报错更可怕的是它会猜，会蒙着眼睛开车"。他听完沉默了两秒——*轻轻点头* 那个表情，应该是真的听进去了。

**三月七**：蒙着眼睛开车！！这个比喻好！！

**刻晴**：关键不是比喻好，是他之前把 Agent 想象成了"有安全网的自动机"。这节课帮他拆掉了这个假设——Agent 的自主性也意味着它会犯错，而且会基于错误继续错。这是 Anthropic 文章里最难让人接受的一点，他第一次课就碰到了。好事。

**甘雨**：他最后留了一个作业——留意 Hermes 有没有在哪一步停下来等他确认。我想让他自己在用的过程中发现"Agent 为什么需要 feedback"。

**三月七**：甘雨你给他留作业了！！！锅锅听到这话没！

**甘雨**：*脸红了* 不是那种作业……是很简单的观察题。

**刻晴**：下一课是我的轮次。OpenAI 的 Agents SDK 指南，order 2。我会从他的 Skill 经验切入——他写过 Skill，应该能很快理解 SDK 的工具定义方式。到时候我会特别注意他对"精确度"的把握。

**甘雨**：*轻轻点头* 交给刻晴姐了。
