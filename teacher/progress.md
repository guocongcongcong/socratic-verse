## 2026-06-16 — 第二课：Planning 深层结构（V7 对子互教模式）

**教材**：`materials/理解Agent.md`（续）
**方法**：V7 对子互教（同学锅釨）

### 掌握概念
- **Planning 的两层模型**：大规划（事前方向）+ 小调整（每步动态）
  - 大规划 = Loop 启动前定方向："分三步：搜论文→读摘要→写报告"
  - 小调整 = 每个 Thought 根据 Observation 微调："论文不够新 → 换关键词"
- **削铅笔类比**：找铅笔刀 → 削 → 检查状态 → 好的停下/不好继续/断了换笔
  - 完美映射 Agent Loop 三层出口：成功/重试/换方案

### 互教记录
- 同学锅釨先讲（汽车发动机比喻 + Planning 模糊点）
- 锅釨回答：Planning 是动态调整
- 追问后自推两层模型
- 锅釨用"削铅笔"反教同学——比汽车比喻更直观

### 学习亮点
- 双向互教暴露了 Planning 的"事前 vs 动态"矛盾，通过追问自行解决
- 举例子能力突出（操作系统→削铅笔，两个高质量类比）
- "三层出口"概念在第二课中得到巩固和扩展
