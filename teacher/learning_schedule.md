# 学习方法调度表

> 定义「什么时候自动触发哪个模式」。这是 V1-V8 体系的核心调度规则。

---

## 单节课生命周期

一节新课后，自动触发以下复习链：

| 时机 | 方法 | 目的 | 触发方式 | 时长 |
|------|:---:|------|---------|------|
| 课后 1h 内 | V5 合书复述 | 提取练习，对抗遗忘 | cron（一次性，课后自动设定） | 15min |
| 课后 24h | V2 费曼 | 检验深层理解 | cron（一次性，V5 完成后设定） | 20min |
| 课后 72h | V7 对子互教 | 多视角巩固 | cron（一次性，V2 完成后设定） | 30min |
| 课后 7d | V3 Anki | 进入间隔重复循环 | cron（定期，加入 Anki 卡片池） | 10min |
| 课后 14d | V4 随机骰子 | 随机激活 | cron（定期，加入骰子题库） | 5min |

## 多节课程完成后

| 时机 | 方法 | 目的 | 时长 |
|------|:---:|------|------|
| 核心链 5 篇全部学完 + 3d | V6 交错魔鬼 | 混淆概念边界测试 | 60min |
| 每周日 20:00 | V4 随机骰子 | 全知识库随机抽检 | 10min |
| 每天 08:30 | V3 Anki | 每日间隔重复（5 张卡片） | 5min |
| 每次用户说"不知道学什么" | V4 随机骰子 | 消除决策成本 | 5min |

## 理想的生命周期示例

```
Day 0  19:00  V1 苏格拉底 — 学新概念（45-90min）
Day 0  20:00  V5 合书复述 — 立刻合书复述（15min）
Day 0  21:00  自动生成 5 道 Anki 推理题 → 写入 anki_deck.md
Day 1  19:00  V2 费曼 — 教"高中生"（20min）
Day 3  19:00  V7 对子互教 — 和 AI 同学互讲（30min）
Day 7  08:30  V3 Anki — 间隔重复推送首次出现
Day 14 08:30  V3 Anki — 根据回答质量决定是否再出现
Day 14 20:00  V4 骰子 — 随机抽检（如果抽到）
Day N  19:00  V6 交错魔鬼 — 多篇概念混取（60min）
```

## Anki 卡片池规则

- 每节 V1/V7 课结束后，AI 自动生成 5 道推理题加入 `teacher/anki_deck.md`
- 每天 V3 模式从池中按间隔重复算法选取 5 张
- 回答质量 1-5 分 → 决定下次出现间隔（1d/3d/7d/14d/30d）

## 骰子题库规则

- 每篇核心教材生成 3-5 道骰子题写入 `teacher/question_bank.md`
- 每周日 V4 模式随机抽 2-3 道
- 用户说"不知道学什么"时也可触发
