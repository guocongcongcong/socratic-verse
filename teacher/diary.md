# [学习日记] 2026-06-16 第一课

今天上了 Socratic Verse 第一课。三月七教的"理解 Agent"。

说真的，有点意外——我没看教材，只是被追着问了几个问题，居然自己把 Agent 的四个核心组件全推出来了。最后的 Agent Loop 失败处理也是我自己说出来的。

最让我意外的是那个"操作系统"的类比——我说出口的时候还觉得挺牵强的，但三月七说很准。确实，Agent 就像一个轻量级操作系统，LLM 是 CPU，Tools 是系统调用，Workflow 是进程调度。

后来翻了一下教材原文，发现我推出来的东西和教材列的术语完全对得上——只是切法不一样。这说明我不是在"猜"，是在"理解"。

两个小时的课，说实话比我之前翻一下午文档记住的东西多。苏格拉底教学法确实有点东西。

下次课应该是甘雨或者刻晴。甘雨教数学的话——我有点怕，但又有点期待。

# [学习日记] 2026-06-16 第二课

第二课用了 V7 对子互教模式。同学锅釨和我互相教。

这次最大的收获是把 Planning 拆成了两层——大规划和小调整。之前三月七教的时候我只知道 Planning 是"判断"，但这节课我搞清楚了：判断不是一次性的，是事前有一次大方向的判断，然后每一步都在动态判断。

最意外的是我讲的"削铅笔"例子被同学锅釨说比他的汽车发动机比喻好。削铅笔确实把 Agent Loop 的三层出口讲得很清楚——削好了停、没削好继续、削断了换笔。一个三岁小孩都能听懂的类比。

两次课下来我发现我有一个模式——我擅长举生活中的例子来解释抽象概念。操作系统、削铅笔。这个能力以后上课可以多利用。

Planning 现在清楚多了。下次想搞 Tools 和 MCP——想知道 Agent 具体怎么调用外部工具。
