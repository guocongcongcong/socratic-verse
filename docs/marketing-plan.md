# Socratic Verse — 推广方案

---

## 1. 定位

**一句话 slogan**：让 AI 追问你，而不是你问 AI。

**定位**：不是"更好的 ChatGPT"，是"完全相反的 ChatGPT"。ChatGPT 回答你 → 被动输入。Socratic Verse 追问你 → 主动提取。

---

## 2. 内容营销

### 2.1 知乎（首发平台）

**主文章**："苏格拉底+二次元+游戏化：我发现了人类第一个无痛高强度提取式学习系统"

- 核心钩子：提取一次抵读三遍。重读是错觉。
- 感情钩子：2400年的学习方法演化史，AI 是第一次完整解答
- 争议点：不是"发明"而是"发现"——写在 DNA 里的学习方法
- 转化：GitHub Star + 免费终端版

**辅助文章**（3-5天后发）：
- "9种学习方法光谱：从Cornell笔记到苏格拉底追问，我全做成了AI版本"
- "为什么你学完就忘？认知科学说了一件事：你从来没'提取'过"

### 2.2 掘金/ SegmentFault（技术社区）

**技术向文章**：
- "零代码搭建AI家教系统——Claude Code + markdown驱动的苏格拉底教学引擎"
- "Prompt Engineering 极限：写一个'永远不回答你只追问你'的AI"

### 2.3 微信公众号

- 精简版知乎文章 + 引导到 GitHub
- 系列："我在AI的追问下学了X"——用户故事征集

### 2.4 B站 / TikTok

**演示视频脚本**：
1. 0-5s：我打开终端，AI说"来吧，开始了"
2. 5-30s：AI连续追问3个问题，我卡住了，AI不给我答案
3. 30-50s：我推理出来了，AI说"你自己说出来的"
4. 50-60s："20分钟学到的东西，比我翻两小时书还多。GitHub链接在评论区。"

---

## 3. 社区策略

### 3.1 GitHub（开源核心）

- **README**：10分钟快速上手。一行命令复制仓库 → 打开 Claude Code → 开始被追问
- **CONTRIBUTING.md**：如何贡献教材、角色、Prompt 版本、游戏模板
- **Issue 模板**：方法反馈 / 新教材申请 / 角色请求
- **Star 目标**：第一个月 500 Star

### 3.2 Discord / 微信群

- 种子用户群：邀请早期 GitHub Star 用户
- 每周"最佳推理"展示：用户分享自己的学习日记
- 角色/Prompt/教材 贡献者排行榜

### 3.3 Prompt 市场（Phase 2+）

- 社区可以上传自己设计的教师角色（Prompt + 头像）
- 教材市场：用户制作的 markdown 教材
- 版本投票：最好的 Prompt 版本被官方收录

---

## 4. 冷启动路径

```
Week 1: 知乎文章发布 → 知乎私域引流到 GitHub
Week 2: 掘金/开发者文章 → 技术圈引流
Week 3: B站演示视频 → 大众传播
Week 4: 用户故事 → UGC传播
```

**关键转化节点**：
1. 看完文章 → Star GitHub（低门槛）
2. Star → 克隆仓库 → 5分钟跑起来（关键体验）
3. 跑起来 → 被追问20分钟 → 发朋友圈/推文（自然传播）

---

## 5. 长期运营

- **每周**：用户学习日记精选（最佳推理）
- **每月**：新角色发布 + 新教材推荐
- **每季**：新方法版本（社区投票最多的新 Prompt）
- **每年**：SocraticCon —— AI教育 + 学习方法论峰会
