# Socratic Verse — 商业企划书

> 2026年6月 | 锅釨

---

## 执行摘要

**Socratic Verse** 是一个基于 AI 苏格拉底教学法的开放学习系统。核心洞察：2400年的学习方法演化史证明，最高效的学习是"提取"——但提取需要意志力，而重读不需要。AI 首次将提取强度和意志力消耗解耦：一个会追问的 AI 伙伴，让你以最低的意志力获得最高的学习效率。

**一句话**：让 AI 追问你，而不是你问 AI。

---

## 1. 市场分析

### 1.1 市场规模

| 市场 | 规模（2026） | CAGR |
|------|:---:|:---:|
| 全球在线教育 | $350B | 12% |
| AI 教育 | $25B | 36% |
| 语言学习 App | $15B | 18% |
| 中国 K12 课外辅导 | $120B | 8% |

**可服务市场（SAM）**：自学编程 + 考研考公 + 终身学习者。中国 ~5000万人，全球 ~2亿人。

### 1.2 市场痛点

1. **大多数人还在用重读**——最无效的学习方法。不是因为他们傻，是因为提取需要意志力。
2. **AI 教育产品都在做"让 AI 回答问题"**——但这是低提取的。真正高效的是"让 AI 追问你"。
3. **现有产品竞争的是碎片时间**（Duolingo 5分钟、得到音频）——没人做**高强度沉浸式学习**。

### 1.3 竞争格局

| 玩家 | 模式 | 提取强度 | 我们的优势 |
|------|------|:---:|------|
| ChatGPT/GPT | 回答式 | ★ | 我们反转：追问式 |
| Duolingo | 游戏化+记忆 | ★★ | 我们做深度推理 |
| Khan Academy | 视频+练习 | ★★ | 我们全程主动 |
| Socratopia | 苏格拉底+二次元 | ★★★★★ | 我们+8种方法+语音+游戏化 |
| Anki | 间隔重复 | ★★★ | 我们+追问+自动生成推理卡 |

**蓝海定位**：Socratopia 证明了市场需求存在。但它是封闭的成品软件，月费订阅。我们是**开放系统**：开源 Prompt + markdown 驱动 + 社区贡献。类似 WordPress vs Wix 的关系。

## 2. 产品

### 2.1 核心功能（Phase 1-4）

| Phase | 内容 | 时间 |
|:---:|------|:---:|
| 1 | 方法论体系 + 知乎文章 + Prompt 系统（开源） | June 2026 ✅ |
| 2 | 文字客户端（终端/Web） | Q3 2026 |
| 3 | 语音模式（TTS+STT 边走边学） | Q4 2026 |
| 4 | 游戏模式（捏脸+知识生成游戏） | Q1 2027 |

### 2.2 差异化

1. **方法多元化**：8种学习方法，用户可选或AI自适应。竞品只有单一方法。
2. **反转交互**：AI追问你，不是你问AI。本质不同的认知负荷。
3. **开放系统**：markdown 驱动，任何人都可以贡献教材、角色、Prompt 版本。
4. **全模态**：文字→语音→游戏，循序渐进，降低接入门槛。

## 3. 商业模式

### 3.1 收入模型

| 层级 | 价格 | 内容 |
|------|:---:|------|
| **免费 / 开源** | ¥0 | 终端版 + 核心 Prompt（Claude Code / Hermes Agent 直接跑） |
| **Pro** | ¥29/月 | Web 客户端 + 语音模式 + 教师角色库 + Anki卡片生成 |
| **Max** | ¥69/月 | 游戏模式 + 捏脸 + 知识游戏生成 + 自定义角色 |
| **教育版** | ¥9/月/学生 | 班级管理 + 教师Dashboard + 学习分析 |

### 3.2 为什么免费+付费能工作

**免费层是漏斗**。终端用户（程序员、AI 爱好者）自然成为传播者。他们在社交平台分享"被 AI 追问 20 分钟学到的东西"，这是最好的广告——因为他们是真的学到了。

**付费层提供便利**。语音模式、Web 客户端、游戏生成——这些是非技术用户需要的。

### 3.3 规模化路径

```
Phase 1: GitHub 开源 → 技术社区传播 → 1000 Star
Phase 2: 知乎/掘金/公众号 方法论文章 → 非技术用户认知 → Web 客户端发布
Phase 3: 语音模式 → TikTok/B站 演示 → 视频传播
Phase 4: 游戏模式 → 教育机构合作 → B2B 销售
```

## 9. 总结

> "我不认为 Socratic Verse 与目前任何 AI 教育产品形成竞争。因为大多数 AI 教育产品的着眼点，是对碎片时间的利用。我们只专精于一条很少有人顾及的学习场景：高强度沉浸式学习。" — 改写自吴乐旻

这不是一个"更好的 ChatGPT"。这是一种**学习范式转换**。从 AI 回答你 → AI 追问你。从被动输入 → 主动提取。从意志力消耗 → 游戏般的沉迷。

**让世界上任何人都能以最低的意志力消耗，获得最高强度的提取式学习体验。** 这句话如果在 2026 年不能实现，那就永远不会实现了——因为 AI 已经到了那个临界点。
